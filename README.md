# #项目简介
该项目是一个仿茶七网的web app全栈项目

前端主要包含以下几个模块：首页、分类、购物车、个人页。

实现的功能：商品详情展示、商品搜索、登录、注册、找回密码、模拟支付、短信验证码、地址等功能

# #技术栈
vue2 node.js axios sass vant-ui 

# #克服的技术难点
1.购物车的数据联动
2.生成订单及连接支付宝沙箱
